Guía para Usuarios – Evolución UCI

Todas las funciones disponibles en el módulo de Evolución de Enfermería

Acceso al Sistema

Inicia sesión con tu usuario y contraseña para proteger tu información clínica.

Crear Informe Nuevo

Selecciona Box, rellena los campos y pulsa “Generar Informe” para guardar uno nuevo.

Recuperar Informes

Elige de la lista tu Box, fecha y hora para cargar y visualizar informes anteriores.

Editar Informe

Modifica los campos de un informe cargado y vuelve a “Generar Informe” para guardarlo con nuevo ID.

Eliminar Informes

Elimina un informe aislado o limpia todos los informes de un Box de forma rápida.

Imprimir Informe

Imprime en turno actual (8–20 h o 20–8 h) o usa “Turno Alternativo” para el turno opuesto.

Copiar Informe

Envía el informe al portapapeles con formato enriquecido para pegar en otros documentos.

Cerrar Sesión

Finaliza tu sesión para proteger tus datos antes de abandonar la aplicación.
